How does it work?

1. ビデオの縮小 (Resize the video)

ビデオを扱いやすい様に48x48に縮小するだけ
Just resize the video to 48x48 for easier handling

2. Canvasへの転写 (Copy to Canvas)

drawImageを使ってビデオをCanvasへ転写する
Copy the video to Canvas with drawImage

3. Data URLの取得 (Get Data URL)

canvas#toDataURLでCanvasをData URLに変換する
Convert Canvas to Data URL with canvas#toDataURL

4. faviconのhrefに指定する

Data URLをfaviconのhrefに指定する
Set href of the favicon with Data URL

That's all!
